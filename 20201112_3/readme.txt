Задача_3. Написать программу, которая заставляет человека вводить координаты вершин треугольника в формате (x1, y1), (x2, y2), (x3, y3), до тех пор, пока это не окажутся координаты вершин треугольника в указанном формате, а потом выводит его площадь с точностью до сотой. Если формат ввода неправильный, программа вместо площади выводит «Invalid input», а если формат правильный, но площадь его нулевая — «Not a triangle».

Написать тесты на неправильный формат
Написать тесты на не-треугольник
Подсказка (x1, y1), (x2, y2), (x3, y3) = eval(input()) спасут немного времени

Пример работы:

asdf
Invalid input
1,2,3,4,5,6
Invalid input
(1,1), (2,2), (11,11)
Not a triangle
(1,2), (4,5), (9,8)
